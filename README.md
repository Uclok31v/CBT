# CBT

## Overvies
This is a four-choice orthodox Computer Based Testing Application.
You can set questions to be asked according to the attributes of the examinees and the number of times they have taken the exam.

## List of Functions
### Common
- Log in
- Log out
### Administrator
- Users (administrators / examinees) registration
- Examinees viewing
- Examinees editing
- Examinees deletion
- Questions registration
- Questions viewing
- Questions editing
- Questions deletion
- Password change
### Examinee
- Problem set construction
- Question
- Answer
- Scoring
- Pass / fail judgment

### Dependency
- Java(Servlet/JSP)
- Bootstrap
- MySQL
